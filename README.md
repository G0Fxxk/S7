# 西门子PLC上位机学习项目 #

本项目是一个基于 WPF 构建的西门子 S7 系列 PLC 通讯监控与可视化分析工具。通过底层的 `S7.Net` 库实现了对 PLC 的高效、稳定交互，提供从基础的变量监控、数据块解析，到复杂的运动轴控制以及实时数据流趋势绘制等全面功能。

## 主要功能模块

### 1. 🌐 核心通讯与连接 (Connection)
- **广泛的型号兼容**：支持通过 TCP/IP 连接西门子  S7-1200 及 S7-1500 等主流设备。
- **全方位健康监控**：内置 ICMP Ping 检测以及心跳轮询，可实时观察通讯延迟与丢包状态。
- **安全与自愈**：支持通讯异常自动重连机制，避免由网络波动导致的停止工作；能够实时读取并显示 PLC 此刻的 CPU 工作状态（Run / Stop）。

### 2. 🎛️ 自定义控制面板 (Control Panel)
- **灵活的变量绑定**：允许用户手动添加感兴趣的 PLC 绝对地址（如 `DB1.DBX0.0`，`MD100` 等），进行高频监控与值修改。
- **丰富的数据类型**：完整支持读取和写入 `Bool` (布尔型), `Int` (16位整数), `DInt` (32位整数), `Real` (单精度浮点) 以及自定长度的 `String` (字符串) 数据。

### 3. 📈 高级实时趋势图表 (Multi-Trend View)
- **高性能采集与渲染**：基于 `ScottPlot` 引擎的图表系统，支持多通道、多数据类型的实时高频数据采集与动态展现。
- **灵活的数据分析**：支持框选放大缩小 (Zoom)、平移 (Pan)、自由开启/关闭的历史滑动窗口 (Auto-Scroll) 以及游标卡尺测量。
- **独占采样模式**：为保障敏感数据（如高速运动脉冲或模拟量抖动）记录的绝对连续性，支持在采样期间挂起其他线程，集中所有通讯资源以提高采样率。
- **导出机制**：支持将绘制的曲线截图与数据值，一键归档保存为 CSV 数据表格或生成 PDF 报表。

### 4. ⚙️ 运动控制与轴调试 (Axis Control)
- **实时控制面板**：专门针对伺服轴和步进电机等运动对象设计的可视化面板，支持读取轴的位置、速度反馈以及异常代码，并下发 JOG、点位移动等指令。
- **参数备份配置**：支持运动控制参数与伺服配置信息的导入与导出，免去在不同设备间重复手动输入的繁琐过程。

### 5. 🗃️ 批量数据块分析与配方管理 (DB Parser & Recipe)
- **数据结构解析**：可直接加载或导入 TIA Portal（博途）导出的数据块及符号信息，并在界面上解析与管理复杂数据。
- **配方系统**：针对工业现场换产需求，实现了基于配方的系统参数存储。支持一键将配方参数块下发至设备 DB 块，或从设备中上传最新的工艺参数进行备份归档。

## 运行与开发环境
- **框架**: .NET 9.0 (WPF 开发，仅运行于 Windows)
- **依赖库**: S7.Net (西门子通信) / ScottPlot.WPF (科学图表绘制) / CommunityToolkit.Mvvm (MVVM 侧边架构)
